How it works

  • Model tabs (GFS/ECMWF/ICON) switch the entire table between models
  • Metric toggle (TWS/SOG/TWA/Swell) changes what the gradient bars represent
  • Bars use CSS gradients sampled every ~1hr along the route for smooth transitions
  • Color scales adjust per metric (e.g., for TWA: green=beam reach, yellow=close hauled, red=no-go zone)
  • Hover a row → route appears on map; click to "select" it as the active route
  • Rows sorted by total passage duration (best first)
